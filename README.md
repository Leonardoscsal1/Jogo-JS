2. Regras do jogo: O jogo não tem nenhuma regra especifica é bem simples e intuitivo, a única regra seria seguir os comandos que é pedido no jogo como por exemplo: "Voce quer continuar s ou nao?" colocar a letra s ou a letra n para indicar o sim ou não. ou por exemplo colocar a letra da alternativa que voce deseja marcar como resposta.


3. Como jogar é bem simples e intuitivo, voce começa digitando seu nome e o jogo vai te mostrar quando de dinheiro voce vai conseguir se voce errar, se voce parar ou se voce acertar e o preço total do jogo, e logo em seguida vai te mostrar a pergunta com as alternativas e voce deve escolher entre uma das alternativas ou entre parar o jogo.

4. Como executar é apenas utilizando o comando npm start

5. Eu utilizei ferramentas como o https://www.w3schools.com/js/ e também a IA do google como por exemplo se voce pesquisar como utilizar  uma funçao em javascript ela vai te mostrar como funciona além de te dar um exmeplo detalhado de maneiras de como aplicar.
